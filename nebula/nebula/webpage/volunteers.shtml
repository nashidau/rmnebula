Projects for Volunteers

As with most projects there are more things to be done in nebula then there are people to do them. Additionally we may not have all the skills for a particular project.

Jonathan Dale's MSExcel Spreadsheet on Gnumeric

The Gnumeric spreadsheet can load and handle a lot of MSExcel documents. Unfortunatly it throws a lot of wanring when loading the spreadsheet. It would be nice to be able to fix it so it works with both Excel and Gnumeric, or failing that so it handles at least gnumeric cleanly.

Gnumeric is avaible at the Gnumeric home page. You can get Jonathan Dale's spreadsheet from Jonathan's roleplaying page.

Nebula on a microsoft platform

Nebula is written in ANSI-C using GTK+2.0 and some Gnome libraries. Nearly everything has been ported to windows, but it would be nice to have nebula ported as well. You will need a compiler on windows, either gcc or maybe a microsoft compiler, and preferably some experience with porting unix apps to windows. We can provide help on nebula itself, can host binaries, and will check in appropriate patchs into the main tree.

Instructions for getting nebula is available on the main page of this site. For GTK+2 on windows see Tor Lillqvist's site. Compilers and other windows binaries are available at the Cygwin page, and their list of ported applications may be helpful. Additionally see Steven Obrien's pages on Gnome.
