Nebula: Features

Updates of individual items appear in brackets - form is YYYYMMDD.

Features

These features are complete. Note in some cases they have associated bugs or similar

  • Character format in XML - (mostly) compatible with ACG!
  • Loading,
  • Saving,
  • Name, Race, class etc.,
  • Stats - all 10 + Appearance, Willpower & Sanity,
  • Stat bonuses,
  • Stat Gain,
  • Skill categories, skills and skill groups,
  • Skill bonus calculations (including stats, ranks, special bonuses, profession bonuses and progression types),
  • Can buy skills,
  • DP calculation (both total, and skill purchasing),
  • Saves skills back to character,
  • Creation:
    • Choose Genre/Setting,
    • Character Name,
    • Choose Race,
    • Choose Profession,
    • Choose Realm (for Non spell users),
    • Choose Religion (for channelers),
    • Choose Essence school (for Essense users),
    • Choose Extra Base lists (for pure spell users)
    • Delete unwanted spell lists,
    • Select weapon costs,
    • Choose Stats,
    • Roll Potentials,
    • Purchase hobby ranks
  • Everyman, Occupational Restricted skill purchase
  • Support restricted half ranks
  • Ability to create a 'Level Template' so you can save the skills you have purchased, and reapply them for later levels
  • Can add skills in right click menu

Unimplemented Features

  • Toggle for extra stats (Sn, Wl, Ap)
  • Equipment,
  • Encumbrance,
  • Training Packages,
  • Talents and Flaws,
  • Addition of categories
  • Logging of changes
