Data Formats

Nebula now supports an XML format in addition to the old format. The old format is now considered deprecated, and although still supported, it may not be in the future. It is recommended you convert all your old format characters to the new XML format. Details of the old format are in old.

Character Format

  • HTML: description of the character format, this is designed as the human reference,
  • DTD: for the character format, electronic reference for the format.
  • CSS: a simple style sheet to render a character,
  • XSLT: a more complicated rendering for the character
