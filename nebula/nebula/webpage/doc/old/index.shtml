Data Formats

Nebula reads a number of formats for various items of data. In all cases they are text files and can be manually editied with any text editor (vi, emacs or Notepad).

All formats are intended to be documented and should appear here. If you have any corrections for any of the files please notify me (click contact to the right).

Current formats:

  • Character Files
  • Genre/Setting Files
  • Profession Files
  • Race Files
