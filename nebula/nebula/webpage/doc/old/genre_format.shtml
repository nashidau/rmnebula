Genre Format

The genere files determine what base skills are availabel for the chracters. It is important to note that:

  • There is no attempt to restrict classes or races based on genre &
  • The genre has no effect after character creation.

It is suggested that GMs provide a genre file for campaigns they run, addign or removing skills and languages as they see fit. In particular they should make sure commonly available languages, lores and any special skills are in the data file.

The format is identical to the skills section of the character files. Obviously you can't have ranks, but it supports things like Everyman progressions, and the same stat bonus format.

The only other significant difference is the use of the stat realm to indicate the realm stat.
